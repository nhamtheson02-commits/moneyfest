# MONEYFEST Project Context

MONEYFEST is a personal finance education and lead generation website.

## Current Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL/Neon
- Zod
- React Hook Form
- Basic Auth protected internal route

## Deployment Direction

- Dev repo: `https://github.com/JasonBui1285/moneyfest`
- Production repo: `https://github.com/nhamtheson02-commits/moneyfest`
- Hosting: Vercel
- Database: Neon PostgreSQL
- Build command: `npm run build`

## Current Scope

Phase 1 includes public pages, ebook lead capture, blog content, tools, services, contact forms, and a basic protected internal dashboard.

Phase 2 admin work is scoped but not implemented in this task.
